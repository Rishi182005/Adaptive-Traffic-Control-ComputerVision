# TrafficSync-AI – Adaptive Traffic Control using Computer Vision

## Overview

TrafficSync-AI is a real-time traffic signal optimization system that uses computer vision to detect vehicles and dynamically adjust traffic signals.

## Features

- Real-time vehicle detection
- Multi-camera input support (IP camera / USB camera)
- Adaptive traffic signal generation
- Countdown timer control

## Architecture

Camera Input → Detection Model → Vehicle Count Engine → Signal Optimization → Output Controller

## Technologies

- Python
- Computer Vision
- OpenCV
- Real-time video processing

## Engineering Highlights

- Real-time inference pipeline
- Modular camera input switching
- Dynamic signal generation logic
